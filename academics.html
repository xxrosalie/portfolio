CSC486

CSC486 is the class Computational Visual Narrative. This was a project-based computer science course that I took in my Junior year. In this class, we created 4 video games within a group of atleast 3 people. Each game had to take place within the same narrative, so, game 1 started the story and game 4 ended the story. However, each game was different in the type of game it was. Game 1 was a narrative game, game 2 was a hidden object game, game 3 was a pinball game, and game 4 was a point-and-click RPG. I was the artist of these games. You can see one piece of art for each of the 4 games to the right. I made much more art, but these are some of my favorites for each game.

  • Tools: IPad, Shetchbook, Godot
  • Skills: Digital art, UI design, Story/narrative writing, Four different types of game mechanics

CSC461

CSC486 is the class Computer Graphics. In this class, we coded 5 projects using the tools discussed below. Projects 1 through 4 were practicing and learning basic computer graphics coding, such as raytracing, rasterization, shaders, textures, and transformations. Project 5 was to build a Battlezone game with the use of all the techniques we learned in projects 1-4. I made two version of this game. Version 1 is the regular arcade vector-graphics styled Battlezone game. The second version used OBJ models instead of vector-graphics. I initially created OBJ models with textures, but coding the textures was not possible within the timeframe I had, so right now you can only see the non-textured OBJ models. My initial idea was to make it a candyland theme, so with lollipops as obstacles, pink tanks, and pink snowy mountains.

  • Tools: JavaScript, WebGL, VSCode, HTML, JSON, BlockBench
  • Skills: Computer Graphics coding, OBJ and JSON models, Raytracing, Rasterization, Textures, Transparency, Shaders, Transformations

GC420

GC420 is the class Visual Thinking. This class was offered to all Computer Science students since visual thinking is an important skill for a computer scientist to have. I personally also loved this class because each homework required drawing, which is one of my hobbies. Each week in this class, we would have to draw something different and use our creativity to make it different than the drawings of other students. Each drawing taught us another visual thinking technique. At the end of the semester, we had to choose one visual thinking technique/topic that we learned about throughout the semester and make a large art piece about it. My final project can be seen here. I chose to make it about the topic "Recentering" which explains that people tend to see every object, person, or concept in a stereotyped way at first, but if you recenter your brain, you can see other versions or aspects of the thing in a more creative way. I tried to show this process with a girl. The stereotyped version of her is drawn on the black paper, then the green "glitches" show the process of recentering how you view this girl which shows that you can also see her in other ways such as having curly hair, freckles, imperfections, etc. Neither view is "correct" but it shows that there are multiple ways of seeing things besides the stereotyped view you have when first thinking of it in your brain.

  • Tools: Sketchbook, Pencil, Pen, Sharpie, Colored pencils, Markers, Paint
  • Skills: Visual Thinking, Drawing, Painting, Creative Thinking
